## Tech Stack
- **Backend**: Django REST Framework, Python
- **Database**: MySQL
- **Frontend**: Next.js with Tailwind CSS
- **AI**: LM Studio (Llama 3.2 1B)
- **Scraping**: BeautifulSoup, Requests
